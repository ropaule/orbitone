# Orbitone 🌀 Webbased Polyrhythm Explorer

Orbitone is an opensource web-app designed to enable quick and intuitive generation of synth based polyrythms. <br>
[This video](https://www.youtube.com/watch?v=lsAlFKXAMjM&list=RDlsAlFKXAMjM&start_radio=1) by virtual sparked my interest for the project.
